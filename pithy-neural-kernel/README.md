# PITHY NEURAL KERNEL V1.1

A real Self-Organizing Map running live in your browser.

One HTML file. Zero dependencies. Pure vanilla JavaScript.

Watch 1,600 neurons compete in real time as they organize random RGB chaos into smooth, evolving color fields. Click and drag to inject color. Toggle toroidal topology. Crank the parameters. Pause the simulation.

This is unsupervised AI training you can see breathing.

No servers. No Python. No frameworks.

Just drop the file and the machine starts thinking.

---

## Features

### Learning Engine
- Real Kohonen Self-Organizing Map (unsupervised learning)
- Live competitive training across a 40x40 neural grid
- Optional toroidal topology (edges wrap seamlessly)

### Interaction
- Click-and-drag color injection for targeted training
- Real-time parameter tuning while the model learns
- Pause and resume the simulation at any moment

### Control and UX
- Draggable cyberpunk control panel
- Fully controllable parameters: learning rate, neighborhood radius, update speed, glow intensity
- Keyboard shortcuts (Space toggles panel, Escape or ` as backup)
- Snap-back panel reset if you drag it into chaos
- Clean, legible UI that survives any color field
- About modal with proper lore

All of it lives inside a single, self-contained index.html.

---

## How to Run

1. Clone or download the repo  
2. Open index.html in any modern browser  
3. Press Space to access the control panel  
4. Click and drag on the canvas to inject colors  

No build step.  
No dependencies.  
No bullshit.
