# Pithy VerLite: Kinetic Neural Kernel v3

A real Verlet Integration cloth physics engine running live in your browser.

One HTML file. Zero dependencies. Pure vanilla JavaScript.

Watch a 22x22 grid of constrained nodes simulate cloth in real time - pinned, wind-blown, and interactive. Launch marbles from the left. Let them tear through the fabric. Detonate them all at once.

This is Newtonian physics you can touch.

No servers. No Python. No frameworks.

Just drop the file and the simulation starts breathing.

---

## Features

### Physics Engine
- Real Verlet Integration, the same math used in movie-grade cloth simulation
- Iterative constraint relaxation across the full node grid, 18 passes per frame
- Drag-to-G correlation: lunar and zero-G modes automatically tune friction for cinematic motion
- Stress-aware rendering: strands glow brighter as they stretch

### Interaction
- Click and drag individual nodes to pull the cloth
- Launch marbles at variable speeds, angles, and entry heights
- Shatter the atmospheric particles with a click
- Detonate all active marbles simultaneously with a shockwave blast

### Control and UX
- Real-time parameter tuning: wind, gravity, snow density, grid size, color mode
- Five visual protocols: Vibrant Rainbow, Hyper Drive, Plasma Vibe, Neon Green, ROYGBV Static
- Live kernel diagnostics: FPS, wind velocity, node count
- Flash effect and audio feedback on key events
- Manifesto modal with proper lore

All of it lives inside a single, self-contained HTML file.

## How to Run

1. Clone or download the repo
2. Open `verlite.html` in any modern browser
3. Click the canvas to interact with the cloth
4. Use the control panel to tune the simulation
